Starting up

The Franka Emika arm should be connected to the shop floor controller. When the controller box is turned on, the arm LEDs will flash yellow. Once booting is complete, the LEDs will turn solid yellow. At this point the brakes can be released.

(image of the wiring) Wiring configuration for using the Franka.

When the arm's LEDs show solid white, the robot is in movement mode. The external activation device (EAD), is a software stop. When pressed, it 'detaches' the arm from the controller which causes the arm to lock in place (the LEDs will show solid yellow when this happens). It is not an emergency stop, as it does not cut power to the arm/controller.

Whilst the Arm is in movement mode, it can be manually manipulated by pressing both of the buttons at the end effector simultaneously. Whilst both buttons are pressed, the arm can be manually moved to a desired position. The buttons have a two stage press, and if you press too hard on the buttons, the arm will lock again. The Arm will automatically go into gravity compensation mode when manually moving it.

Removing or adding the Franka's end-effector (hand)

To detach or reattach Panda's end effector, the arm must be shut down completely first. Then, the end effector can be added or removed by removing/replacing both screws and attaching/detaching the interface cable.

Once the robot has restarted, go to Settings in the web interface, go to End effector and set both the hand drop-down menu and toggle the gripper.

Franka Emika Software

Software updates can be found at: http://support.franka.de/

The software versions currently used in the robotics lab are:

Franka Firmware 1.0.9

Supports libfranka < 0.2.0

ros-kinetic-libfranka 0.1.0

Current ROS version is 0.2.0

The lab only supports libfranka 0.1.0 which is currently unavailable from apt install. Do NOT uninstall ROS or libfranka on workstations which already have it installed.
